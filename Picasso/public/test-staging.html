Staging Widget Test

Test Checklist (Security Phase 3 — P19)

Tenant:

  1. Widget opens/closes — state persists across open/close
  2. Send a message — streamed response comes back
  3. CTA buttons render and trigger actions
  4. External link CTAs open in new tab (allow-popups)
  5. Start a form, close chat, reopen — form state persists in session

If all 5 pass, the iframe sandbox + sessionStorage migration is solid.
