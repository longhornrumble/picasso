Welcome to Page 1

This is the first page of our persistence test. Use the chat widget to start a conversation, then navigate to other pages using the links above to see the conversation persist.

The chat widget uses sessionStorage to maintain:

  • Your conversation history
  • The session ID
  • Widget open/closed state
  • Scroll position

Session timeout is set to 30 minutes of inactivity.
